Setup :
1. cp .env.example .env
2. composer install
3. php artisan serve

How to use Frame Panel command :
1. Register Controller : framereg:controller {namespace}
2. Register URL : framereg:url {function name} {method (GET, POST, PUT, DELETE)} {controller ID}
